# Personal Productivity Suite

A modular, CLI-based productivity suite for Python internship project submissions.

## Features
- Calculator with basic arithmetic, advanced ops and history
- Notes Manager with add/view/search/edit/delete/export
- Timer & Stopwatch
- File Organizer (move & group by extension, preview files)
- Unit Converter (length, weight, temperature)
- Backup & Restore (data/backups folder)


## Setup
1. Python 3.9+
2. `python -m venv venv` (optional)
3. `pip install -r requirements.txt`
4. `python main.py`


## Project structure
See repository root files and `modules/` for implementation.


## Tests
Run `pytest tests/`.


## Notes
- Data files stored under `data/`
- Keep regular backups using the Backup menu
